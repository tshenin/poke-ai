Privacy Policy

Poke AI — last updated March 3, 2026

What the extension does

Poke AI is a productivity tool that lets engineers launch Cursor background agents directly from Jira tickets. It reads ticket information from the active Jira page and sends it to the Cursor API.

Data collected

The extension accesses the following data, all of which stays on your device or is sent only to the Cursor API:

  • Jira ticket content – ticket key, title, and description are read from the currently open Jira page when you trigger an agent.
  • Cursor API key – your personal API key is stored locally in Chrome's extension storage (chrome.storage.local).
  • Settings – repository URL, branch/ref, and custom prompt templates are stored locally in Chrome's extension storage.

How data is used

  • Jira ticket data is combined with your prompt template and sent to api.cursor.com to start a background agent.
  • Your API key is sent to api.cursor.com as an authentication credential.
  • All settings are used exclusively to configure agent requests. Nothing is used for analytics, advertising, or profiling.

Third-party services

The extension communicates only with api.cursor.com. No data is sent to any other third-party service. There are no analytics, tracking pixels, or telemetry of any kind.

Data storage & retention

  • All data is stored locally on your device using chrome.storage.local.
  • No data is stored on external servers controlled by the extension developer.
  • Uninstalling the extension removes all stored data.

Data sharing

Your data is not sold, shared, or transferred to any third party, except as described above (Cursor API for agent execution).

Permissions explained

  • storage – to persist your API key and settings locally.
  • activeTab – to read ticket information from the currently active Jira tab when you click the extension.
  • host_permissions (*.atlassian.net) – to run the content script that extracts Jira ticket data.
  • host_permissions (api.cursor.com) – to send agent launch requests to the Cursor API.

Changes to this policy

If this policy is updated, the updated date at the top will be changed. Since this is an internal tool, changes will be communicated through normal team channels.

Contact

For questions about this extension or its data practices, reach out to the maintainer.
