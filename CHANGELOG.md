# Changelog

## [0.13.0] - 2026-03-09

### Added
- On-Site Annotation feature for platform pages
- Annotation mode to select UI elements and add notes for fixes/changes
- Run cloud agent directly from platform pages with annotations
- Visual markers show annotated elements on page
- Annotations included in agent prompts (both cloud and local)

## [0.12.0] - 2026-03-06

### Added
- Model chooser in settings - select which AI model to use for cloud agents
- Models are fetched from Cursor API and saved to preferences

## [0.11.0] - 2026-03-06

### Added
- Clickable "Open agent" link after successful cloud agent run
